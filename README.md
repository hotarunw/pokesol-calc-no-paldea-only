# pokesol-calc-no-paldea-only

[![Static Badge](https://img.shields.io/badge/github-repo-blue?logo=github)](https://github.com/hotarunw/pokesol-calc-no-paldea-only)
[![Static Badge](https://img.shields.io/badge/GreasyFork-install-orange?logo=greasyfork)](https://greasyfork.org/ja/scripts/574342-pokesol-calc-no-paldea-only)

ポケソルダメージ計算/ポケモン管理サイトで、パルデアポケモンのみのトグルを自動でオフにする。

## 対応ページの例

- <https://sv.pokesol.com/calc>
- <https://sv.pokesol.com/team-builder>

## 備考

team-builderの方は、`onlyPaldeaPokemon=false`のクエリパラメータをつければオフにできるが、calcはできないぽい🥐。
